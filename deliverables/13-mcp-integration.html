MCP & AI INTEGRATION

8 MIN READ

WHAT IS MCP

Model Context Protocol (MCP) lets AI assistants talk directly to your CRM. Instead of logging into HubSpot, clicking through menus, and manually pulling up data, you ask a question in plain English and get an answer instantly.

Think of it as giving your AI assistant (like Claude, ChatGPT, or Copilot) a secure pass to read and update your HubSpot data. It can look up contacts, check deal status, find open tickets, and even create new records — all through conversation.

HUBSPOT'S MCP SERVER

HubSpot provides two types of MCP servers:

REMOTE SERVER

CRM DATA ACCESS

Connects any MCP-compatible AI tool to your CRM. Read and write access to your customer data.

Read & Write: Contacts, Companies, Deals, Tickets, Products, Invoices, Quotes, Subscriptions, Orders, Line Items

Read Only: Campaigns, Metrics, Landing Pages, Website Pages, Blog Posts, Users, Teams, Roles

DEVELOPER SERVER

CLI-BASED TOOLS

For developers building on HubSpot. Helps scaffold projects, manage apps, and test integrations through the command line.

USE CASES PER TEAM

Here's what each team can do with MCP after consolidation — just by asking their AI assistant:

SALES
DEAL INTELLIGENCE

"Show me all deals closing this month over $10K"

Instantly see pipeline value, deal stages, and contact history — no clicking through filters.

MARKETING
CAMPAIGN ANALYTICS

"What's the open rate on our Product A newsletter this quarter?"

Pull campaign metrics across brands without switching between dashboards.

SERVICE
CUSTOMER CONTEXT

"Find all open tickets for Company X across all products"

Get the full picture before calling a customer — every product, every ticket, one answer.

LEADERSHIP
EXECUTIVE REPORTING

"Revenue by product line this quarter vs last quarter"

Real-time executive summaries without waiting for someone to build a report.

HOW MCP CHANGES AFTER CONSOLIDATION

BEFORE (MULTI-PORTAL)

  • 4 separate MCP connections needed
  • Each connection only sees one portal's data
  • "Find all tickets for Company X" returns incomplete results
  • Cross-product queries impossible

AFTER (UNIFIED PORTAL)

  • 1 MCP connection
  • Complete view of all customer data
  • "Find all tickets for Company X" shows everything
  • "Revenue by product line" works natively

SETUP OVERVIEW

Setting up MCP requires:

  1. Create a user-level app in HubSpot with the appropriate read/write scopes
  2. Authenticate via OAuth 2.0 (transitioning to OAuth 2.1)
  3. Connect your MCP-compatible AI client (Claude, ChatGPT, etc.)
  4. Admins control which data the AI can access through scope permissions
AVAILABILITY

HubSpot's MCP server is currently in public beta. Enterprise accounts get the most complete data access. Sensitive data properties (PHI, highly sensitive fields) are excluded from MCP access for security.
