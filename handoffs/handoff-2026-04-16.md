### Features Built
- **Vertical scroll timeline** with reveal animations (12 steps + ongoing)
- **Breadcrumb navigation** (frosted glass, BBL blog pattern)
- **Sticky sidebar TOC** with auto-generated scroll-spy from h2/h3 headings
- **Step-through prev/next buttons** (emerald green, hover effects)
- **Dark/light mode toggle** (persists via localStorage, button in nav + mobile menu)
- **Hamburger mobile menu** (full-screen overlay, colour-coded sections, theme toggle)
- **Mobile responsive** (tables scroll, content contained, logo scales, burger menu)
- **BBL branding** (Inter font, blue-tinted borders, emerald/sky/gold accents, transparent logo)
- **Edward as Project Lead** on every timeline step
- **Copyright notice** on every page footer
- **"What to Expect" callouts** on every migration phase
